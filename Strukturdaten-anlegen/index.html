    Strukturdaten anlegen

    Allgemeines

    Strukturdaten können auf unterschiedliche Weise angelegt werden:

    1. Im Strukturbaum mit der rechten Maustaste das Strukturelement anklicken, unter dem das neue Element erstellt werden soll und "Element hinzufügen" auswählen. Es öffnet sich ein Fenster, in dem der Strukturtyp und die zugewiesenen Seiten ausgewählt werden können.
    2. In der Galerie-Ansicht die Images auswählen, die einem Strukturelement zugewiesen werden sollen. Mit der rechten Maustaste das Menü "Element hinzufügen und ausgewählte Medien zuweisen" anzeigen und auswählen. Es öffnet sich ein Fenster, in dem der Strukturtyp und die zugewiesenen Seiten ausgewählt werden können.

    ℹ️ Hinweis: Bei der Variante 1 wird in einigen Fällen die Image-Nummerierung angepasst, wodurch die ursprüngliche Reihenfolge nicht mehr rekonstruierbar ist.

    ℹ️ Tipp: In dem Menü zur Auswahl des Strukturelements kann der Anfangsbuchstabe des Elements eingegeben werden, um rascher das Element zu finden. In dem Menü zur Auswahl der Seiten kann die Imagenummer und die Paginierungsnummer eingetragen werden.

    Anlegen im Strukturbaum

    Mit Rechts-Klick auf des Strukturelement, von dem aus ein neues Strukturelement angelegt werden soll, wird das Menü aufgerufen.

    Danach wird die Position und der Typ des neuen Strukturelements festgelegt.

    ℹ️ Tipp: Nach dem Aufklappen des Menüs werden mit Eingabe des Anfangsbuchstabens des Elements (E für Einband) alle Elemente mit dem Anfangsbuchstaben nacheinander angezeigt.

    Danach werden dem Strukturelement die zugehörigen Seiten zugewiesen.

    Danach wird das Strukturelement im Strukturbaum und in der Bild-Galerie angezeigt.

    Anlegen in der Galerie

    Strukturelemente können auch über die Galerie erstellt werden. Die Auswahl der Images erfolgt über:

    • "Strg" + Mausklick → Auswahl mehrerer einzelner Images
    • Mausklick erstes Image / "Umschalt" (Shift) + Mausklick letztes Image → Auswahl mehrerer zusammenhängender Images

    Beispiele

    ℹ️ Hinweis: Die Dateigröße der nachfolgenden zwei Beispiele musste auf Grund von Limitierungen durch Github verkleinert werden, dadurch kommt es zu Artefakten und Unschärfen. Wir bitten dies zu entschuldigen.

    Verknüpfung eines Images an zwei Strukturelemente

    Ein Image kann nur einem Strukturelement zugewiesen werden. Wenn auf einem Image ein Strukturelement endet und ein neues Strukturelement beginnt, kann eine Verknüpfung erstellt werden. Dies erfolgt in folgenden Schritten:

    1. Das Strukturelement mit dem "geteilten" Image anlegen
    2. Das folgende Strukturelement ohne das "geteilte" Image anlegen (ansonsten wird es nur dem zweiten Strukturelement zugewiesen)
    3. Mit rechter Maustaste das "geteilte" Image im Strukturbaum auswählen und Verknüpfung einfügen

    Ein Image kann auch mehreren Strukturelementen zugewiesen werden. Zum Beispiel, falls sich mehrere Abbildungen auf einer Seite befinden.
